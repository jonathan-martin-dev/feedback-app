# Feedback Application

This is a simple CRUD app that allows users to:

- Send feedback
- Edit feedback
- Delete feedback
- Rate their experience
- See the average rating based on feedback
- See the amount of reviews

### Technologies

- React (Hooks)
- React Router
- Context API
- db.json
